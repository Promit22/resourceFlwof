About ResourceFlow

The ResourceFlow API allows users to post, update, delete, and retrieve links to shared content.

API Endpoints

  • GET /links: Retrieve all available links : https://resourceflwo.onrender.com/links
  • POST /links: Add a new link: https://resourceflwo.onrender.com/links
  • PUT /links/:id: Update a specific link: https://resourceflwo.onrender.com/links/:id

Overview

GET request: you can retrive all the links without an api key.

However, for POST & PUT you must send your valid api key along with the request.

You can get your api key from here.

You can also interact with the resourceFlwo api using GUI, the easiest way to use resourcesFlow API. Just don't forget about the key!

Getting All the Links


      // Example: GET request to retrieve all links
      fetch('https://resourceflwo.onrender.com/links', {
      method: 'GET',
      headers: {
      'Content-Type': 'application/json'
      }
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));

Posting a Link


      // Example: POST request to post your link
      const body = {
      sharedBy: name, //'John Wick'
      type: //eg. sci-fi/educational/entertainment etc,
      description: //a short description about the resource you are sharing
      fetch('https://resourceflwo.onrender.com/links', {
      method: 'POST',
      headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'your-api-key-here'
      },
      body: JSON.stringify(body)
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));

Updating a Link(the link you have posted)


      // Example: Update request to update your link
      fetch('https://resourceflwo.onrender.com/links/:id', { //id as a number eg. 1,2 the specific link you want to update
      method: 'PUT',
      headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'your-api-key-here'
      },
      body: "Your desired link(as a object) with the type and link(must) and optional sharedBy and description property" //make sure to JSON.stringify() it
      })
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error fetching links:', error));
Note: response time can vary between 50sec to 1 minutes due to the usages of render's free service.
